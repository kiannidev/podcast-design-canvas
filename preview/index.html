Podcast Design Canvas

Follow the core episode path from imported speaker files through cleanup, captions, and export readiness. Each step opens a working prototype screen.

  1. 1. Check source media health Review imported speaker files before styling.
  2. 2. Repair speaker sync Fix timing between separate speaker recordings.
  3. 3. Clean up audio Balance sound with plain-language quality choices.
  4. 4. Review caption quality Confirm captions stay trustworthy after audio changes.
  5. 5. Review export readiness Confirm the episode is ready to publish.

Start a local server from the repo root, then open /preview/. Example: python3 -m http.server 8080
