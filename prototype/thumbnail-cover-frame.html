Export Consequences

Only one active cover Promoting a new frame automatically demotes the previous active cover to alternate instead of duplicating publish targets.
Warnings stay attached to a frame An alternate can keep readability or crop concerns even while it stays out of the publish slot.
Missing still blocks required destinations If no active cover is chosen for a destination that requires one, export readiness should keep the thumbnail item blocked.

Connected workflow

  • brand layout stays tied to show brand kit inputs
  • crop safety stays aligned with destination preview surfaces
  • selected cover flows into metadata, review copy, and export handoff
  • preview concerns stay visible before the frame becomes publish-ready
